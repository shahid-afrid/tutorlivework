## ? THE MAGIC: Why It's Dynamic

### Old Way (Hardcoded):
```csharp
[HttpGet]
public IActionResult CSEDSDashboard()
{
    var students = _context.Students.Where(s => s.Department == "CSEDS").ToList();
    // Hardcoded for CSEDS only!
}

[HttpGet]
public IActionResult ECEDashboard()
{
    var students = _context.Students.Where(s => s.Department == "ECE").ToList();
    // Need separate method for each department!
}

[HttpGet]
public IActionResult EEEDashboard()
{
    var students = _context.Students.Where(s => s.Department == "EEE").ToList();
    // Repeat for every department!
}
```

### New Way (Dynamic):
```csharp
[HttpGet]
public async Task<IActionResult> GetDynamicDashboard()
{
    var department = HttpContext.Session.GetString("AdminDepartment");
    var normalizedDept = DepartmentNormalizer.Normalize(department);
    
    var students = await _context.Students
        .Where(s => s.Department == normalizedDept)
        .ToListAsync();
    
    // Works for ALL departments automatically!
}
```
