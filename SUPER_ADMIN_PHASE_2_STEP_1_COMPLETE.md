## ?? **Technical Details**

### **Dashboard Data Flow**
```
User Login
    ?
SuperAdminController.Dashboard()
    ?
SuperAdminService.GetDashboardData()
    ?
- GetAllDepartmentsWithStats()
- GetSystemStatistics()
- GetRecentActivities(20)
    ?
SuperAdminDashboardViewModel
    ?
Dashboard.cshtml
    ?
Rendered HTML with stats
```

### **Statistics Calculated**
- **TotalDepartments**: COUNT from Departments table
- **ActiveDepartments**: COUNT where IsActive = true
- **TotalStudents**: COUNT from Students table
- **TotalFaculty**: COUNT from Faculties table
- **TotalSubjects**: COUNT from Subjects table
- **TotalAdmins**: COUNT from Admins table
- **TotalEnrollments**: COUNT from StudentEnrollments table
- **TodayEnrollments**: COUNT where EnrolledAt.Date = Today
- **ActiveSchedules**: COUNT from FacultySelectionSchedules where IsEnabled = true
