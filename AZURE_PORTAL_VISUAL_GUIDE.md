## ?? TIPS FOR NAVIGATION

### Finding Resources:
1. Use the **Search bar** at the top (??)
2. Type resource name and press Enter
3. Or go to **All resources** to see everything

### Common Issues:
- Can't find something? Use Search!
- Page not loading? Refresh browser
- Changes not applying? Click **Save** and **Continue**
- Still not working? Restart the web app

### Important Buttons:
- ?? **Save** - Always click this after changes!
- ?? **Restart** - Restart web app after config changes
- ?? **Get publish profile** - Download deployment credentials
- ??? **Delete** - Be careful with this!

---

## ?? QUICK CHECKLIST

As you work through the portal, check off:

- [ ] Resource group created (green checkmark appears)
- [ ] SQL Database deploying (wait for notification)
- [ ] SQL Database ready (notification says "Deployment succeeded")
- [ ] Connection string copied to Notepad
- [ ] Web App deploying (wait for notification)
- [ ] Web App ready (notification says "Deployment succeeded")
- [ ] Connection string added to Web App config
- [ ] Application settings added (4 settings)
- [ ] Clicked Save after each config change
- [ ] Publish profile downloaded

---

**Remember:** Azure Portal auto-saves most things, but always click the **?? Save** button when you see it!
